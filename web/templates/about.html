About Pâri Pakûru'

Pâri Pakûru' is a digital dictionary and language reference tool for the Skiri Pawnee language, built to support language preservation and revitalization efforts.

Data Sources

  • A Dictionary of Skiri Pawnee by Douglas R. Parks — the primary linguistic reference, containing over 4,200 headwords with phonetic transcriptions, grammatical classifications, etymologies, paradigmatic verb forms, and example sentences.
  • Pâri Pakûru' (Blue Book) — a Skiri Pawnee language textbook used in community language classes, providing attested vocabulary and usage in pedagogical contexts.

Search & Dictionary

  • Bidirectional search: English → Skiri and Skiri → English
  • Smart search ranking: exact matches first, Blue Book–attested terms boosted, example sentence snippets surfaced
  • Multiple pronunciation displays: IPA phonetic, simplified respelling with pitch accent highlighting
  • Spelling preference toggle: Parks linguistic orthography or simplified spelling
  • Full verb conjugation tables for documented paradigms
  • Semantic categorization and tag browsing
  • Blue Book attestation badges and confidence scores for data quality
  • Etymology, morpheme breakdowns, and cognate cross-references to related Caddoan languages

Learning Tools

  • Weekly flashcard study system with 21 categories and curated beginner vocabulary
  • Template-based sentence builder for practicing Skiri Pawnee word order
  • Interactive pronunciation guide with respelling symbols and pitch accent legend
  • Suggested learning path for new learners
  • Learner / Expert mode toggle to control detail level
  • Dashboard with personal study overview and dictionary statistics

Community & Data Quality

  • Community feedback: flag uncertain entries or confirm correct ones, with admin review
  • Confidence scoring: each entry carries a weighted score based on attestation, completeness, and source agreement

This tool presents data extracted from published linguistic sources. Forms marked as "attested" appear directly in the Parks Dictionary or Blue Book. Any generated or computed forms are clearly distinguished. For an endangered language, every verified form is precious — treat unverified forms with appropriate caution.

Technology

Built with Python, Flask, and SQLite. The dictionary database includes full-text search indexes for fast lookup. The interface uses Pico CSS for responsive, mobile-friendly design and HTMX for live search. Accessibility features include keyboard navigation, collapsible entry sections, and screen-reader-friendly markup.
