About Pari Pakuru

Pari Pakuru is a digital dictionary and language reference tool for the Skiri Pawnee language, built to support language preservation and revitalization efforts.

Data Sources

  • A Dictionary of Skiri Pawnee by Douglas R. Parks — the primary linguistic reference, containing over 4,200 headwords with phonetic transcriptions, grammatical classifications, etymologies, paradigmatic verb forms, and example sentences.
  • Pari Pakuru (Blue Book) — a Skiri Pawnee language textbook used in community language classes, providing attested vocabulary and usage in pedagogical contexts.

Features

  • Bidirectional search: English to Skiri and Skiri to English
  • Multiple pronunciation guides: IPA phonetic, simplified respelling
  • Full verb conjugation tables for documented paradigms
  • Semantic categorization and browsing
  • Blue Book attestation badges marking terms used in teaching materials
  • Etymology and morpheme breakdowns
  • Cognate cross-references to related Caddoan languages

Important Note on Data Quality

This tool presents data extracted from published linguistic sources. Forms marked as "attested" appear directly in the Parks Dictionary or Blue Book. Any generated or computed forms are clearly distinguished. For an endangered language, every verified form is precious — treat unverified forms with appropriate caution.

Technology

Built with Python, Flask, and SQLite. The dictionary database includes full-text search indexes for fast lookup. The interface uses Pico CSS for responsive design and HTMX for live search.
