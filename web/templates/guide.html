Skiri Pawnee uses a small set of sounds. Below you'll find the Parks notation (used in the dictionary headwords), our simplified respelling (shown throughout this site), and English approximations to help with pronunciation.

Vowels

Pawnee has four basic vowel qualities. Each can be short or long (held about twice as long). Vowel length changes meaning, so pay attention to it. {% if spelling_pref != 'parks' %} In the simplified spelling, long vowels use a circumflex: â, î, û. {% else %} In Parks notation, long vowels are simply doubled: aa, ii, uu. {% endif %}

Note: u vs {% if spelling_pref == 'parks' %}uu{% else %}û{% endif %} in pronunciation

In the pronunciation guide (e.g. PIHT-koo), both short u and long {% if spelling_pref == 'parks' %}uu{% else %}û{% endif %} are written as "oo" — English doesn't have a standard way to show this difference. To tell them apart, check the spelling: {% if spelling_pref == 'parks' %} u = short, uu = long (held longer). {% else %} u = short, û (circumflex) = long (held longer). {% endif %} For example: pitku has short u, while {% if spelling_pref == 'parks' %}aruusa{% else %}arûsa{% endif %} (horse) has long {% if spelling_pref == 'parks' %}uu{% else %}û{% endif %}.

About the middle vowel e

The Blue Book (Pâri Pakûru') uses e in some spellings. This sound sits between i and a — the tongue is relaxed but higher than for a. In the Parks dictionary, this is represented by the IPA symbols ə (schwa) or ɪ, and it's respelled as a or i depending on context.

Whispered Vowels

At the end of a word, after a hard consonant (p, t, k, s, c), vowels are often whispered — barely voiced, more like a puff of breath. In fluent speech they may be almost inaudible. This is normal and not a mistake.

Accent Marks

Some forms in the dictionary carry accent marks (á, í, ú). These indicate high pitch on that syllable — Pawnee uses pitch rather than stress to mark emphasis. In the respelling, accents are not shown, but they can affect meaning.

Syllable Structure

In the dictionary's phonetic transcription, a dot (•) marks syllable boundaries, and a dash (–) marks stem boundaries. For example:

[•rə–hʊh–kaa–paa–kɪs•]
duh-hooh-kah-pah-kihs

Quick Reference: Reading the Dictionary

Each entry in this dictionary shows up to three forms:

  1. Headword — the Parks dictionary spelling (e.g., taraha)
  2. Simplified — a learner-friendly respelling with circumflex long vowels (e.g., taraha — same here since no long vowels)
  3. IPA — the precise phonetic transcription in square brackets

{% if spelling_pref != 'parks' %} The simplified spelling uses â for long a, î for long i, û for long u, and ' for the glottal stop. {% else %} The Parks spelling uses doubled vowels (aa, ii, uu) for length and ʔ for the glottal stop. {% endif %} When in doubt, the IPA form is the most precise guide.
